# Provider thumbnails

Đặt bốn ảnh WebP tại đây với đúng tên:

- `piper.webp`
- `genvoice.webp`
- `elevenlabs.webp`
- `minimax.webp`

Khuyến nghị ảnh vuông, tối thiểu 256 × 256 px. Giao diện sẽ crop ảnh bằng `object-fit: cover`.
Nếu ảnh chưa tồn tại hoặc tải lỗi, thẻ tự động dùng ký tự viết tắt hiện tại.
